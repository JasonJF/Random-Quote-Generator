Random Quote Generator

Hi, I'm Jason.

This Random Quote Generator is a mini project that I've created to learn about full-stack development as well as API's.

Here are some of the tech I used in this site:

Bootstrap

I decided to use Bootstrap to make it easier for me to make the site responsive.

Node.js and Express

For the back-end I used Node.js and express to handle things such as API calls.

Heroku

To deploy my app on the web I made use of Heroku

API

The API that I used to generate the random quotes comes from https://github.com/lukePeavey/quotable.
